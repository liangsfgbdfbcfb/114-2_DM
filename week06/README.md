=== 課程摘要 ===

本週課程介紹了人類探索海底世界所使用的各種科技技術，以及海洋探測資料的蒐集與應用方式。內容涵蓋聲納系統、遙控無人載具（ROV）、自主式水下載具（AUV）等海洋探測工具，說明其在海底地形測繪與海洋研究中的重要性。課程也提到海底探測過程中會產生大量數據，必須透過資料分析與探勘技術才能有效利用。透過這些科技的發展，人類得以更深入了解海洋環境與海底地貌。

=== 學習心得 ===

Q1：人類用了哪些科技手段來「看見」海底？
請列舉至少兩種技術，並簡述各自的優缺點。

A1：

人類探索海底最常使用的技術之一是聲納系統（Sonar），例如單音束與多音束測深儀。它利用聲波反射原理測量海底深度與地形，能夠快速覆蓋大範圍海域，適合進行海底地圖繪製。不過聲納測量結果容易受到海水聲速變化影響，因此需要額外進行校正。

另一種技術是遙控無人載具（ROV）。ROV可以攜帶攝影機及各種感測器進入深海環境，提供即時影像與環境資訊。其優點是能直接觀察海底狀況，缺點則是操作成本較高，且活動範圍受到纜線長度限制。此外，自主式水下載具（AUV）也逐漸普及，能夠自動執行任務，但設備成本與維護需求較高。

Q2：這些海底探測技術會產生大量的數據資料。
你覺得這些資料適合用哪些資料探勘方法來分析？請舉例說明。

A2：

海底探測技術所蒐集的資料包含海底地形、水深、聲速、影像及環境參數等，資料量相當龐大。我認為首先可以利用聚類分析（Clustering）將不同區域的海底地形特徵進行分類，例如區分海溝、海脊或平坦海床等地形。

另外，也可以利用影像辨識與深度學習技術分析ROV或AUV拍攝的海底照片，自動辨識珊瑚礁、生物群聚或人工設施。若要預測環境變化，則可以使用迴歸分析或時間序列分析，研究海水溫度、潮位及海底地形隨時間變化的趨勢。這些方法都能幫助研究人員從大量資料中找出有價值的資訊。

Q3：綜合第 4、5、6 週三次協同教學的內容，
你對海洋資料科學有什麼整體的認識或感想？
這些內容如何影響你對期末專題方向的思考？

A3：

透過第4、5、6週的協同教學，我了解到海洋資料科學是一門結合海洋觀測技術、資料蒐集、資料處理以及分析應用的跨領域學科。從潮位觀測、聲速剖面分析到海底探測技術，都需要透過大量數據與資訊科技來提升研究效率與精度。過去我以為海洋研究主要依靠實地觀測，但現在發現資料分析與人工智慧技術已成為重要工具。

這些課程也讓我開始思考期末專題可以結合資料科學方法進行海洋相關分析。例如利用公開海洋資料建立潮位預測模型，或分析海洋環境因子與測深資料之間的關聯性。透過資料探勘與視覺化技術，不僅能提升分析效率，也能讓研究成果更容易被理解與應用。
